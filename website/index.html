Ready to Start?

TailorFit AI is currently in Developer Beta. Follow these quick steps to add it to Chrome:

  1. Download or Clone the GitHub Repo.
  2. Run npm install && npm run build in your terminal.
  3. Go to chrome://extensions in Chrome.
  4. Turn on Developer Mode.
  5. Click Load Unpacked and select the dist folder.
